What is CSS Hole?

CSS Hole is a showcase of static web pages that provide interactive elements without any Javascript. Using only HTML and CSS, you can do any of the following:

  • Play sudoku

  • Plot your journey around the United States

  • Create your own electoral map

  • Create your own U.S Congressional District map

  • Learn about binary

  • Test your knowledge of trigonometric values
